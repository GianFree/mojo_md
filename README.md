# Vision 

Create a working md engine that simulates in NVE and NVT an ideal gas.

## Compilation
To compile the code:
```bash
mojo build main.mojo 
```

## Testing
To run the tests in `test/`:
```bash
mojo test test/test_mojo_md.mojo
```
